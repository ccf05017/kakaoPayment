# kakaoPayment
카카오페이 사전과제 프로젝트(20200513)

## 요구사항
- CardInfo
    - [X] 카드번호(long), 유효기간(int), cvc(int)로 구성된다.
    - [X] 카드번호는 10 ~ 16자리 숫자이다
    - [X] 유효기간은 4자리 숫자이다
        - [ ] 기능 구현 끝난 뒤 날짜 형식 validation 기능 구현(요구사항에는 없음)
    - [X] cvc는 3자리 숫자이다.
    - [X] 암복호화가 가능해야 한다.
        - [ ] 하드코딩 된 키값 설정으로 뺄 것
    - [X] 각각의 속성에 접근할 수 있어야 한다.
    - [X] 동등성 비교를 지원해야 한다.
